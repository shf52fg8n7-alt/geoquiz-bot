# 🌍 GeoQuiz Telegram Bot

## 🚀 ЗАПУСК ЗА 15 МИНУТ

---

### Шаг 1 — Получи токен бота
1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot`
3. Придумай название: `GeoQuiz`
4. Придумай username: `geoquiz_ваше_имя_bot`
5. Скопируй токен вида `7123456789:AAF...`

---

### Шаг 2 — Загрузи на GitHub
1. Зайди на **github.com**
2. Нажми **"New repository"** → назови `geoquiz-bot`
3. Загрузи все файлы этой папки
4. Нажми **Commit changes**

---

### Шаг 3 — Запусти на Railway (бесплатно)
1. Зайди на **railway.app**
2. Войди через GitHub
3. **"New Project"** → **"Deploy from GitHub repo"**
4. Выбери репозиторий `geoquiz-bot`
5. Перейди в **Variables** и добавь:
   - `BOT_TOKEN` = твой токен от BotFather
   - `ADMIN_ID` = твой Telegram ID (узнай у @userinfobot)
6. Railway автоматически запустит бота!

---

## 💰 КАК ЗАРАБАТЫВАТЬ

### Telegram Stars (встроено в бота)
- Premium подписка: **99 Stars** (~$1.3)
- Stars выводятся через Fragment.com

### Реклама
- В коде есть заглушки `show_ad()`
- Подключи **Telegram Ad Platform** когда наберёшь 1000+ подписчиков

### Спонсорство
- При 500+ активных игроках пиши брендам напрямую
